Validação da Implementação do Servidor com cURL

Introdução

Este tutorial orienta você sobre como validar a implementação do servidor utilizando cURL diretamente do terminal. Os comandos fornecidos são essenciais para verificar o funcionamento das rotas e garantir que as funcionalidades foram implementadas corretamente.

Nota: Certifique-se de que o servidor esteja rodando antes de testar. Utilize o seguinte comando para iniciar o servidor: 
node server.js

1. Acessando a Página Inicial (GET /)

Para verificar se a página principal do servidor está sendo exibida corretamente, execute o seguinte comando no terminal:

curl http://localhost:3000/

Esperado: O servidor deve retornar o conteúdo da página index.html.

2. Acessando a Página Sobre (GET /about)

Para testar se a página "Sobre" está acessível e sendo servida corretamente, utilize o seguinte comando:

curl http://localhost:3000/about

Esperado: O servidor deve retornar o conteúdo da página about.html.

3. Simulando Upload de Arquivo (POST /upload)

Para testar a funcionalidade de upload, envie uma requisição POST com dados simulados:

curl -X POST http://localhost:3000/upload -d "fileData=simulacao"

Esperado: O servidor deve retornar a mensagem: "Upload simulado com sucesso!"

4. Testando Rota Inexistente (GET /rota-inexistente)

Para validar o tratamento de erros 404, tente acessar uma rota inexistente:

curl http://localhost:3000/rota-inexistente

Esperado: O servidor deve retornar a página de erro error404.html.

5. Testando Rota Inexistente com POST (POST /rota-inexistente)

Para garantir que o erro 404 também funciona para requisições POST, execute:

curl -X POST http://localhost:3000/rota-inexistente

Esperado: O servidor deve retornar a página de erro error404.html ou uma mensagem informando que a rota não foi encontrada.

Dica: Todos os testes são feitos localmente, então você deve ter certeza de que o servidor está rodando no localhost e na porta correta. Se o servidor estiver em outra máquina ou hospedado online, substitua localhost pelo IP ou domínio.
